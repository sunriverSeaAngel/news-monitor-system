API Online

News Monitor API

Сервис мониторинга новостей с RSS-парсером, RAG-ассистентом, распознаванием речи и Telegram-ботом.

Новости
GET
/api/news
Список новостей с пагинацией и фильтрацией по тегу и источнику.
?page=1 ?limit=20 ?tag=политика ?sourceId=1
GET
/api/tags
Все доступные теги: политика, экономика, технологии, спорт, общество.
AI-ассистент
POST
/api/chat
RAG-ассистент: находит 5 похожих новостей по косинусному расстоянию (pgvector) и отвечает на вопрос через GPT-4o-mini.
{ question: string } → { answer, sources }
POST
/api/speech-to-text
Транскрипция аудио через OpenAI Whisper-1. Принимает файл multipart/form-data в поле audio.
mp3 / wav / webm / ogg / flac → { text: string }
Пользователи и аналитика
POST
/api/users
Регистрация пользователя. Принимает { id, firstName, lastName, email }.
GET
/api/analytics
Аналитика за последние 7 дней: количество новостей по дням, топ-5 тегов, воронка событий пользователей.
Сервисные эндпоинты
POST
/api/telegram/set-chat-id
Привязывает Telegram chat_id к пользователю.
{ userId, chatId }
GET
/api/auth/profile
Профиль авторизованного пользователя (Replit Auth).
GET
/api/healthz
Health-check эндпоинт. Возвращает { status: "ok" }.
